ES5: every、some 判断

every——严格判断

判断数组中的所有元素是否都符合此要求 
            var arr = [1,2,3]
            console.log(arr.every(function(item,i,arr){
                return item > 2
            }))  // false

some——浅判断

判断一个元素是否符合此要求 
            var arr = [1,2,3]
            console.log(arr.some(function(item,i,arr){
                return item > 2
            }))  // true
ES5: forEach 循环

等同于for循环

代替for循环 
            var arr = [1,2,3]
            arr.forEach(function(item,i,arr1){
                arr1[i] = item+1   //循环时执行的操作
            })
            console.log(arr) // 2,3,4
ES5: filter 过滤

复制出原数组中符合条件的元素组成新数组返回，原数组保持不变 
            var arr = [1,2,3]
            var arr3 = arr.filter(function(item,i,value){
                return item > 2   //过滤条件
            })
            console.log(arr3) //3
ES5: reduce 汇总

对数组中的元素进行统计，得出最终结论。原数组保持不变 
            // 汇总  reduce
            var sum = arr.reduce(function(prev,item,i,arr){
                if(item < 3){
                    prev+=item
                }
                return prev   //prev 为起始值，可以是任意值
            },0)
            console.log(sum)  //3
delete和splice删除数组中的元素的区别

参考原文：https://www.cnblogs.com/xielong/p/10120270.html

    1）delete: 只是被删除的元素变成了 undefined 其他的元素的键值还是不变。
            var array=["aa","dd","cc","aa"];
            delete array[1];  //结果：["aa",undefined,"cc","aa"]
    2) splice: 该方法会改变原始数组
            array.splice(1,1);  //会返回所删除的数组
            //输出结果：["aa","cc","aa"]
